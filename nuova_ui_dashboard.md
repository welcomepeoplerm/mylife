
# Nuova UI Dashboard - Istruzioni di Implementazione

## 1. Struttura generale
- AppBar superiore con stile Material 3
- Sezione Hero compatta
- Lista di card verticali con icone e descrizioni

## 2. Gerarchia degli elementi
1. **Top App Bar**
2. **Hero Section**
3. **Card funzionali verticali**

## 3. Linee guida tecniche
### AppBar
- Altezza: 64px
- Elevation: 2
- Icona lingua lato destro

### Hero Section
- Altezza: 120px
- Testo titolo in `title-large`

### Card
- Altezza: 72–82px
- Elevation: 1–2
- Layout: icona a sinistra, testo a destra, chevron finale

## 4. Palette colori
- Primario: #2F8B57
- Secondario: #88C39C
- Sfondo: #F6F8F7

## 5. Tipografia Material 3
- Title Large: 22px
- Body Medium: 14px
- Label Medium: 12px

## 6. Breakpoints
- Layout mobile-first (360px+)
- 1 colonna verticale

## 7. Accessibilità
- Touch target minimo: 48px
- Contrasto minimo 4.5:1
